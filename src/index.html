获取 Cursor Token

操作步骤

1

点击按钮打开 Cursor 登录页

点击下方按钮将在浏览器中打开 Cursor 官方登录页面

2

登录你的 Cursor 账号

在打开的页面中使用你的邮箱、Google 或 GitHub 账号登录

3

获取 Token

登录成功后，按 F12 打开开发者工具

  • 方式一：切换到"应用程序"（Application）或"存储"（Storage）标签
  • 展开左侧 Cookie → https://authenticator.cursor.sh
  • 找到名为 WorkosCursorSessionToken 的 Cookie，复制其值
  • 方式二：在控制台(Console)中输入 document.cookie
  • 找到包含 user_ 开头的 Token，复制整个值
提示：Token 通常很长，确保完整复制。如果无法找到 Cookie，可以在控制台输入 document.cookie 查看所有 Cookie
4

粘贴 Token 并添加账号

将复制的 Token 粘贴到下方输入框中

补丁管理

应用补丁后，当Cursor检测到 ERROR_RATE_LIMITED_CHANGEABLE 错误时， 将自动从配置的API获取新账号并切换，整个过程无需重启Cursor。

注意：应用补丁前请先关闭Cursor。补丁会修改Cursor的核心文件，更新Cursor后需要重新应用。
