🎮 使用指南
基础使用

1. 手动换号
   打开软件，进入"仪表盘"
   点击"输入 Token 一键换号"
   输入您的 AccessToken
   等待换号完成
2. 通过 SessionToken 添加账号
   点击导航菜单"获取令牌"
   点击"打开登录页"
   登录您的账号
   按 F12 打开开发者工具
   找到 Cookie
   复制并粘贴到输入框
   点击"解析并添加账号"
3. 账号管理
   进入"账号管理"页面
   查看所有已保存的账号
   点击"一键换号"快速切换
   使用"导出账号数组"备份账号数据
4. 自动换号
   点击"自动一键换号"
   自动执行换号
   ❓ 常见问题
   Q1: 换号后 Cursor 无法启动？
   A: 尝试以下步骤：

进入"配置备份"页面
选择最近的备份
点击"恢复"
如果问题依然存在，手动删除 Cursor 配置文件后重装
Q2: 提示"Cursor 数据库不存在"？
A: 确保：

Cursor 已正确安装
至少登录过一次 Cursor
Cursor 完全关闭（检查任务管理器）
Q3: 自动换号提示"已过期"？
A:

检查激活码是否在有效期内
Q4: 用量统计显示不准确？
A:

点击仪表盘的"刷新"按钮
确保 Cursor 已登录
检查网络连接是否正常
